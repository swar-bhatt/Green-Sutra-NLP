# **App Name**: Krishi Sahayak

## Core Features:

- Crop Recommendation: Suggest the optimal crop for the next planting cycle, using Gemini AI based on the farmer's location, soil type, previous crop, fertilizer usage, water hardness, available resources, temperature, and rainfall data. The tool will analyze historical data and current conditions to provide personalized recommendations.
- Resource Optimization: Offer advice on how to best utilize limited resources such as water and fertilizer, based on the chosen crop and the farmer's specific circumstances.
- Weather Integration: Integrate live weather data from OpenWeather API to provide real-time information about temperature and rainfall in the farmer's location.
- Disease Prediction & Prevention: Predict potential crop diseases based on environmental factors and suggest preventative measures, using historical data.
- Chatbot Assistance: Provide a chatbot interface for farmers to ask questions and receive immediate assistance in their local language using Google Translate, powered by NLP.
- Multi-language Support: Enable content translation via Google Translate.
- Personalized Dashboard: A personalized dashboard displays relevant information, such as weather updates, crop recommendations, and resource optimization tips, tailored to the farmer's profile and location.

## Style Guidelines:

- Primary color: A lively light-grass green (#C0FF33) evoking freshness and growth.
- Background color: A desaturated, light, closely related green (#E4F7B5).
- Accent color: A vibrant yellow (#FFF233), which is analogous to the primary and complementary to darker hues; the accent is for calls to action.
- Body and headline font: 'PT Sans' for a balance of modernity and warmth, making it easy to read.
- Use a set of simple, clear icons to represent different crops, resources, and weather conditions, providing visual cues for farmers.
- Design a clean and intuitive layout that is easy to navigate, with a focus on displaying key information prominently.
- Incorporate subtle animations and transitions to provide feedback to user interactions, such as loading new data or displaying recommendations.
